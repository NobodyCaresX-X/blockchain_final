# 基于区块链的众筹系统

## 快速启动

### 模式一：Hardhat 外部链 + Metamask（推荐测试环境）

#### 步骤 1：启动 Hardhat 本地区块链

打开终端，进入项目目录，启动 Hardhat 节点：

```bash
cd hardhat
npx hardhat node
```

保持此窗口运行，Hardhat 会在 `http://127.0.0.1:8545` 启动测试链，并生成20个测试账户（每个账户有10000 ETH）。

#### 步骤 2：配置 Metamask 钱包

1. 打开浏览器，点击 Metamask 插件
2. 点击右上角「网络」下拉菜单 → 选择「添加网络」→「添加自定义网络」
3. 填写以下信息：
   - **网络名称**: Hardhat Local
   - **RPC URL**: http://127.0.0.1:8545
   - **链 ID**: 31337
   - **货币符号**: ETH
   - **区块浏览器 URL**: （留空）
4. 点击「保存」添加网络

#### 步骤 3：导入测试账户到 Metamask

1. 在 Hardhat 启动输出中，复制第一个账户的私钥（如：`0xac0974bec39a17e36ba4a6b4d238ff948bacb478cbed5efcae784d7bf4f2ff80`）
2. 在 Metamask 中点击「导入账户」
3. 粘贴私钥，点击「导入」
4. 即可看到该账户余额为 10000 ETH

#### 步骤 4：部署智能合约

打开新终端窗口：

```bash
conda activate ai_env
python deploy_to_local_chain.py
```

脚本会自动将合约部署到 Hardhat 测试链，并输出合约地址。

#### 步骤 5：启动后端服务

```bash
python start_external_chain_mode.py
```

访问：http://127.0.0.1:8000

### 模式二：Python 内置测试链（快速演示）

无需启动外部区块链，一键启动：

```bash
conda activate ai_env
python start_tester_mode.py
```

访问：http://127.0.0.1:8000

> **注意**：此模式使用 Python 内置的 eth_tester 测试链，不支持 Metamask 连接。

## 功能说明

| 功能 | 说明 |
|------|------|
| 项目创建 | 提交项目名称、描述、目标金额、截止日期、阶段点 |
| 资金捐赠 | 向项目捐赠，记录捐赠者和金额，支持内置余额支付 |
| 项目展示 | 显示所有进行中和已结束的项目 |
| 自动结束 | 到达截止日期后自动结束项目，计算阶段点保留金额 |
| 提取资金 | 成功项目发起人提取资金 |
| 退款 | 失败项目捐赠者按比例取回资金，优先使用内置余额 |
| 阶段点 | 阶段性资金释放，项目失败时已达标阶段点自动保留 |
| 里程碑 | 发起者承诺回报内容，无资金释放功能 |
| 早期奖励 | 前10位捐赠者获得特殊标识 |
| 月度支持 | 支持按月自动扣款的长期捐赠模式 |
| 时间同步 | 链上时间与系统时间自动同步，前端自动调整截止时间 |
| 双钱包 | 内置余额用于众筹，外置钱包用于充值提现 |
| Metamask | 支持Metamask连接、签名登录、地址关联 |

## 时间同步注意事项

### Hardhat测试链时间特性

1. **时间只能向前推进**：Hardhat测试链使用 `evm_increaseTime` 只能将时间向前调整，无法向后调整
2. **链上时间可能超前**：如果之前执行过时间调整操作，链上时间可能比系统时间快
3. **自动同步机制**：系统会在每次交易前自动同步链上时间与系统时间

### 截止时间调整

当链上时间比系统时间快时，前端会自动调整截止时间：
- 检测链上时间与系统时间差异
- 如果截止时间在链上已过期，自动延长截止时间
- 确保用户设置的5分钟后截止时间在链上有效

### 手动同步

如需手动同步链上时间，可以调用：
```bash
curl -X POST http://127.0.0.1:8000/api/sync-time
```

## 阶段点资金释放规则

阶段点资金释放基于**目标金额**而非实际筹集金额：

| 场景 | 阶段点达标 | 释放金额计算 |
|------|-----------|-------------|
| 目标100ETH，阶段点60%/20% | 已达标 | 100ETH × 20% = 20ETH |
| 目标100ETH，阶段点80%/40% | 已达标 | 100ETH × 40% = 40ETH |
| 实际筹集90ETH（失败） | - | 总保留 = 60ETH，可退款30ETH |
| 实际筹集100+ETH（成功） | - | 所有资金保留给创建者 |
