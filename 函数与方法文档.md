## AuthService 类方法 (auth.py)

### 用户管理

| 方法名 | 参数 | 返回值 | 功能说明 |
|--------|------|--------|----------|
| `_hash_password` | `password: str` | `str` | 使用SHA256哈希算法加密用户密码 |
| `_initialize_default_users` | 无 | `None` | 初始化默认用户（user01-user10），为每个用户分配随机外置钱包余额（50-500 ETH） |
| `register` | `username: str, password: str, nickname: str = ""` | `Optional[User]` | 注册新用户，创建区块链账户，从默认账户转账100 ETH用于支付gas |
| `login` | `username: str, password: str` | `Optional[str]` | 用户登录，验证密码后返回JWT令牌 |
| `logout` | `token: str` | `None` | 用户注销，删除会话令牌 |
| `validate_token` | `token: str` | `Optional[User]` | 验证JWT令牌，返回对应的用户对象 |
| `get_user_by_address` | `address: str` | `Optional[User]` | 根据区块链地址查找用户 |
| `get_user_by_username` | `username: str` | `Optional[User]` | 根据用户名查找用户 |
| `list_users` | 无 | `list[dict]` | 返回所有用户的列表（不含密码） |

### 双钱包管理

| 方法名 | 参数 | 返回值 | 功能说明 |
|--------|------|--------|----------|
| `topup` | `username: str, amount: float` | `dict` | 充值：从外置钱包转入指定金额到内置余额 |
| `withdraw` | `username: str, amount: float` | `dict` | 提现：从内置余额转出指定金额到外置钱包 |
| `use_internal_balance` | `username: str, amount: float` | `dict` | 使用内置余额（用于捐赠），扣除指定金额 |
| `refund_to_internal` | `username: str, amount: float` | `dict` | 退款到内置余额，增加指定金额 |
| `add_to_internal` | `username: str, amount: float` | `dict` | 增加内置余额（用于里程碑资金释放等） |
| `check_refund_capability` | `username: str, required_amount: float` | `dict` | 检查退款能力：内置余额不足时，检查外置钱包是否足够覆盖退款金额 |

### Metamask 集成

| 方法名 | 参数 | 返回值 | 功能说明 |
|--------|------|--------|----------|
| `get_metamask_nonce` | `address: str` | `dict` | 获取Metamask登录随机数和当前时间戳，用于生成签名消息 |
| `verify_metamask_signature` | `address: str, signature: str` | `bool` | 验证Metamask签名，检查签名是否与随机数消息匹配 |
| `login_with_metamask` | `address: str, signature: str` | `Optional[str]` | 使用Metamask签名登录，验证通过后返回JWT令牌 |
| `get_user_by_metamask_address` | `address: str` | `Optional[User]` | 根据Metamask地址查找已关联的用户 |
| `link_metamask_address` | `username: str, metamask_address: str` | `dict` | 将Metamask地址与系统账户关联 |

## 总结

本系统实现了以下核心功能：

1. **双钱包系统** - 区分内置余额和外置钱包，支持充值、提现、捐赠、退款
2. **用户认证** - 支持账号密码登录和Metamask签名登录
3. **众筹管理** - 创建项目、捐赠、里程碑管理、自动结束
4. **区块链集成** - 使用Web3.py与智能合约交互
5. **Metamask集成** - 支持Metamask连接、签名认证、地址关联

系统采用前后端分离架构，后端使用FastAPI提供RESTful API，前端使用原生JavaScript实现动态交互，智能合约使用Solidity编写并部署到区块链上。
